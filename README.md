# Bus-Eignungstraining

Statische Übungswebseite zur Vorbereitung auf eine Eignungsuntersuchung für eine Busfahrer-Qualifizierung.

## Inhalt

Die Seite trainiert realitätsnah, aber nicht identisch zu offiziellen Testsystemen:

- Deutschverständnis von Anweisungen
- Reaktionsfähigkeit
- Aufmerksamkeit / visuelle Suche
- Konzentration unter Zeitdruck
- Orientierung im Raster / Linienverfolgung
- Logik und Arbeitsgedächtnis
- Belastbarkeit bei mehreren Regeln

## Arabische Hilfe

Bei jeder Aufgabe kann eine arabische Hilfe geöffnet werden. Sobald diese Hilfe genutzt wird, zählt die Aufgabe mit 0 Punkten. So kann die Seite zunächst als Lernhilfe genutzt werden; das eigentliche Ziel bleibt aber, die deutschen Aufgaben ohne Übersetzung zu verstehen.

## Speicherung

Die Testergebnisse werden im Browser per `localStorage` gespeichert. Es werden keine Daten an einen Server gesendet.

## GitHub Pages

Diese Dateien können direkt in ein neues GitHub Repository kopiert werden. Danach unter **Settings → Pages** als Source `Deploy from a branch`, Branch `main`, Ordner `/root` auswählen.
